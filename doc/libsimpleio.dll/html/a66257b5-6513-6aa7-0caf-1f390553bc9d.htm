Wio-E5 LoRa Transceiver Module Amateur Radio Unicast Flavor #1 Services
Classes
 ClassDescription
Device Encapsulates the Wio-E5 LoRa Transceiver Module in P2P (Peer to Peer or Point to Point) broadcast mode with Amateur Radio Unicast Address Header Flavor #1:

8 bytes network ID aka callsign, left justified and space padded.
1 byte destination node ID (ARCNET style: 0=broadcast or 1 to 255).
1 byte source node ID (ARCNET style: 1 to 255).
