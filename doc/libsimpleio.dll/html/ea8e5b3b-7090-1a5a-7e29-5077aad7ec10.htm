kind Returns the kind of mikroBUS shield that is installed on the target board, as obtained from the SHIELDNAME environment variable or guessed from the BOARDNAME environment variable. The guessed value for BeagleBone family target boards (BOARDNAME == "BeagleBone*") is BeagleBoneClick4. The guessed value for Raspberry Pi family target boards (BOARDNAME == "RaspberryPi*") is PiClick3.
