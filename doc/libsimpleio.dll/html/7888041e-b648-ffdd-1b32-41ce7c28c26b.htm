Remarks
The following device tree overlay commands must be added to /boot/config.txt:
C#
Copy
OVERLAYS=MUNTS-0018
The Orange Pi Zero 2W does not route PWM signals to GPIO18 or GPIO19, so Servo Header J2 and DC Motor Driver Grove Connector J7 are not usable except as extro GPIO pins.
